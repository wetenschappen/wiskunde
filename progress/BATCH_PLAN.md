# Batch Improvement Plan

## Strategy

Improve activities in curriculum-module order. Each batch targets all activities supporting a Nando module.

**Core improvement pattern applied to every activity:**
1. **Predict → Set → Verify** — Force commitment before interaction; slider confirms, not discovers
2. **Multi-stage levels** — Break each level into 3-4 cognitive stages (explore, practice, apply, reflect)  
3. **Error analysis** — Detect the 2-3 most common misconceptions per topic
4. **"Why" reflection** — After success, a 2-3 sentence conceptual explanation
5. **Worked examples** — Model the thinking before the student attempts
6. **Verbalization** — Force description in words at key moments
7. **Knowledge proof** — Final level acts as LPD verification
